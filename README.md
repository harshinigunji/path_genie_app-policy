# Path Genie – Education Stream Advisor App

Path Genie is an education guidance application that helps students explore suitable education streams and career paths after key academic stages.

This repository contains both frontend and backend code, organized into separate folders for better clarity and maintenance.

## Project Structure

- **MyApplication**  
  Contains all frontend code, including UI, user interaction, and data display.

- **path_genie**  
  Contains all backend code, including business logic, APIs, and data processing.

## Additional Files

- **privacy-policy.html** – Describes how user data is handled.
- **delete-data.html** – Explains how users can request data deletion.

This project is intended for educational and informational purposes only.
